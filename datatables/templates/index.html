Datatables Example - Upload and Export

This is a simple example of the Datatables plug-in for jQuery. You can upload any .xlsx, .csv, .txt, or .json file and it will display the data here. Any column can be sorted, rearranged, or toggled, and there's a universal search bar that will update the results as you type. You can export the table to a number of formats, csv, xlsx, pdf, copy it to your clipboard, or send it to a printer if you have one. I also made a similar example using Stock Market data, link here
